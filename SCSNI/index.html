SCSNI@UCI

Specialty Care Safety Net Initiative @ UC Irvine

This is a temporary information page regarding the SCSNI Project at UCI. It is for internal use only. Do not share this page with external parties, including clinics and CCPH.

Important Links

Referrals : This is a semi-public Google Spreadsheet used to input specialty requests and their clinic of origin, information needed for both Technical and Program coordinators. It does not contain patient identifiable information. If you have a Google Account please ask Hector for edit access.

Scheduling Calendar : This is a public Google Calendar listing specialty blocks. If you have a Google Account please ask Hector for edit access.

UCI Contacts : This is a semi-public Google Spreadsheet containing non-sensitive UCI Doctor & Staff contact information.

Phase 1 Clinic Information : This is a private Google Spreadsheet containing sensitive SCSNI Phase 1 clinic statistics and technical information. Ask Hector for access.

UCIMC IT : This is a private Google Spreadsheet containing sensitive UCIMC technical information and testing history. Ask Hector for access.
